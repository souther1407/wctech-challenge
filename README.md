# REPOSITORIO DEL CHALLENGE PARA COPYWRITE

## Descripcion
challenge para CopyWrite que consiste en una api que retorna un texto invertido y un front que permite
al usuario ingresar un texto para que la api lo invierta, mostrando todos los resultados en el front


## Como probar

### En la Carpeta Back

- ejecutar el comando npm i
- ejecutar el comando npm i -D
- ejecutar el comando npm run dev
- dejar funcionando el servidor

### En la Carpeta Front

- ejecutar npm i --legacy-peer-deps
- ejecutar npm i -D --legacy-peer-deps
- ejecutar npm run dev
- en la direccion que indica, entrar con ella en el navegador
